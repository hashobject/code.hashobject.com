hashids

A Clojure wrapper library for Hashids.

Install

[hashobject/hashids "0.2.0"]

Usage

user=> (use 'hashids.core)
nil
user=> (encrypt 134 "super-secret-salt")
"Lzn"
user=> (decrypt "Lzn" "super-secret-salt")
134
user=> (encrypt 225 "super-secret-salt")
"7jR"
